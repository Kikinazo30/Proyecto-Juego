# [Duelo Montañoso]

Juego de tipo Metroidvania con modo multijugador cooperativo inspirado en Celeste. Estamos explorando la posibilidad de un pvp o exploracion cooperativa

- W/A/S/D: Movimiento

- Mouse: Apuntar
